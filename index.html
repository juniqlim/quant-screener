공식 요약

DCF (Discounted Cash Flow)

FCF = 영업활동현금흐름 - CAPEX
적정가 = (미래 FCF의 현재가치 합 + 잔존가치) - 부채 + 현금) / 발행주식수
괴리율 = (적정가 - 현재가) / 현재가

DCF 한계

  • 모든 종목에 동일한 FCF 성장률(5%)을 적용 — 종목별 실제 성장성 미반영
  • 과거 확정 실적 기반 FCF 사용 — Forward(26E) 컨센서스 미반영
  • WACC 10% 일괄 적용 — 종목별 자본비용 차이 미반영
  • 사실상 현재 FCF 대비 시가총액이 싼 종목을 찾는 간이 스크리너

Magic Formula (조엘 그린블라트)

ROIC = EBIT / 투하자본 (투하자본 = 유동자산 - 유동부채 - 현금 + 유형자산)
EY = EBIT / EV (EV = 시가총액 + 부채 - 현금)
순위 = ROIC 순위 + EY 순위 (낮을수록 상위)
